# Suggested Commands for Kitaplik App

## Development Commands

- `npm run dev`: Starts the Expo development server. Use this for daily development.
- `npm run android`: Runs the app on a connected Android device or emulator.
- `npm run ios`: Runs the app on a connected iOS device or simulator.
- `npm run build:web`: Exports the app for web deployment.

## Quality & Maintenance

- `npm run lint`: Runs the linter to check for code style issues.
- `npm test`: Runs the Jest test suite.
- `npm run sonar:scan`: Performs a SonarQube scan for code quality analysis.

## Utilities (Windows System)

- `dir`: List files in a directory.
- `git status`: Check current git state.
- `findstr`: Search for patterns in files (Windows equivalent of grep).
- `type [filename]`: Display file content (Windows equivalent of cat).
